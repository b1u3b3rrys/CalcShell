# CalcShell
its calculator which simulates shell and thats all
## Building
### GCC:
```
git clone https://github.com/b1u3b3rrys/CalcShell
cd CalcShell
gcc calc.c -o calc
./calc
```
### Make:
```
git clone https://github.com/b1u3b3rrys/CalcShell
cd CalcShell
make
./calc
```
